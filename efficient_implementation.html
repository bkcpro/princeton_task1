Question 4:

How does the runtime change if the array needs to be sorted first?

If the array needs to be sorted first, a quick sort could be used, but inefficiently as the worst-case time complexity is O(n^2). Alternatively, a Balanced Binary-Search-Tree could be constructed out of the unsorted array and the search could be made. The complexity would be O(n) [for construction - recursively done] + (log n) [for search] ~= O(n)
